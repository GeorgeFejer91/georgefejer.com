# Neutral Hand Audio AI Notes

Canonical page: `/Research/study-6/neutral-hand-audio/`

Legacy aliases: `/audio-assets/`, `/projects/study6-neutral-hand-audio/`,
`/research-projects/study6-neutral-hand-audio/`

Start with `../AI_HANDOFF.md` for the operational handoff, then use
`../protocol/ASSET_AND_WORDING_MAP.md` for the research-facing file map.

These are the audio scripts participants hear during the experiment. There are
four variants of essentially the same instruction per language, with movement
clusters in different orders. Randomly assign those four variants across the
four blocks for each participant/session.

All AI/operator-only notes, prompt experiments, one-off validation harnesses, and
temporary helpers for this audio package should live in this `for-ai/` folder or
a child folder inside it. The reproducible public audio rebuild scripts remain
in `../scripts/` because they are part of the handoff package itself.
