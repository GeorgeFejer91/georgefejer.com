# Project Goals

## Primary Goal

Create a standardized five-minute neutral hand-based task for Study 6. Participants perform simple, non-emotional hand movements while the environment, particles, or hand mapping can be manipulated independently.

## Design Goals

- Keep the task emotionally neutral.
- Avoid symbolic gestures, expressive gestures, fidgeting, self-touch, and social/tactile regulation.
- Keep manual load low enough that the task does not become a performance challenge.
- Provide direct participant-facing cues for when to start, continue, switch, and stop each movement type.
- Standardize total duration, segment duration, cluster duration, and final file duration across English, German, and four version orders.
- Preserve enough movement to keep participants engaged and to support hand/particle mapping manipulations.

## Deliverables

- Four English and four German final MP3 files.
- Exact timed transcripts and wording libraries.
- Final duration, cluster duration, and prompt timing validation reports.
- Cached prompt audio so the final files can be rebuilt without new ElevenLabs calls.
- Rebuild scripts and API-key handling documentation.
- AI handoff documentation for continuation from another GitHub project.
