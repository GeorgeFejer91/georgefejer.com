# Study 6 Legacy ElevenLabs Audio Archive

This folder preserves the Study 6 neutral hand-task audio package that was live before the 2026-07-01 XTTS-v2 conversational rewrite.

Use the parent `../../audio/` folder for the active participant-facing audio. This archive is retained for reproducibility only.

Archived contents:

- `audio/`: previous ElevenLabs MP3 files.
- `transcripts/`: previous timed transcripts.
- `validation_reports/`: previous duration and cue timing validation reports.
- `elevenlabs_render_manifest.json`: previous render metadata.
- `timing_library.csv`: previous cue timing library.
- `ASSET_MANIFEST.csv` and `RESEARCH_PROJECT_MANIFEST.csv`: previous inventory files.
