# Study 6 Research Project

This folder groups the browser-facing and printable assets for the Study 6
research project.

- `quest-sam-questionnaire-preview/` contains the Quest/SAM browser preview and
  SAM pictographic assets.
- `questionnaire-packets/` contains printable questionnaire packets and spatial
  frame PDF blocks.
- `vr-atemexperiment-teilnahmebestaetigung/` contains the VR Atemexperiment
  participation confirmation PDF and preview images.
- `../../projects/study6-neutral-hand-audio/` contains the public Study 6
  neutral hand movement audio guides.

Keep new Study 6 research assets inside this folder so the repository root stays
organized around `Academic/`, `Hobby/`, and `Research/`.
